Certifications DataCamp

Data Scientist Professional & Associate

J'ai obtenu 2 certifications sur le site Datacamp: Data Scientist Associate puis Data Scientist Professional.
Lors de ces 2 examens j'ai dû répondre à un besoin métier spécifique par certification, travail à réaliser dans un temps imparti :

  • Savoir quel article mettre en avant sur un site pour maximiser le traffic
  • Identifier le nombre de personnes qui sont susceptibles d'intégrer un cursus universitaire afin de dimensionner les effectifs

Pour ce faire voici les différentes étapes du projet :

  • Analyser et valider les données : remplacer les valeurs manquantes
  • Visualiser les données : scatterplot, boxplot, corrélation entre les variables
  • Implémenter des modèles de Machine Learning
    • Savoir si un article va générer de bons traffics : Classification
      • Logistic Regression
      • XGBoost Classifier
    • Prédire le nombre de personnes qui vont intégrer le cursus : Régression
      • Linear Regression
      • RandomForest
  • Analyser les performances de chaque modèle avec les bonnes métriques : RMSE pour la régression et précision pour la classification
  • Conclure sur le meilleur modèle suivant le problème et analyser les besoins métiers
  • Fournir des recommendations sur la suite des implémentations et comment maintenir le modèle de ML efficace
  • Présenter les résultats avec un diaporama
